---

## 2. Topic Area

---

## 3. Problem Statement
- Many people in need struggle to find reliable platforms to raise funds and receive support.  
- Donors often lack transparency and trust when contributing to charitable causes.  
- Existing solutions do not always provide proper verification or clear tracking of donations.  
- This creates a gap between people who need help and those willing to provide it.  

---

## 4. Proposed Solution
- Charity Platform is a web-based system that connects donors, people in need, and volunteers in one ecosystem.  
- The platform provides project verification to ensure trust and reliability.  
- It enables transparent donation tracking for all users.  
- An interactive map helps users discover nearby charity initiatives.  
- Secure authentication is implemented using JWT and Spring Security.  

---

## 5. Target Users
- Donors  
- People in need  
- Volunteers  
- Administrators  

---

## 6. Technology Stack

- **Frontend:** Vue.js, Vite, JavaScript  
- **Backend:** Spring Boot, Spring Security, Java 17  
- **Database:** PostgreSQL (Supabase)  
- **Cloud / Hosting:** Render (backend), Netlify (frontend)  
- **APIs / Integrations:** REST API, Map integration, Supabase  
- **Other Tools:** Maven, MapStruct, Git  

---

## 7. Key Features
- User authentication with JWT  
- Role-based access control (Admin, Donor, User, Volunteer)  
- Project creation and verification system  
- Donation management and tracking system  
- Interactive city map for project discovery  
- Admin moderation and control panel
